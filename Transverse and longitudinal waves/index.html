Transverse & Longitudinal Waves

Select a type of wave to simulate.

Transverse Wave

Longitudinal Wave

Select either 'transverse' or 'longitudinal' to see a simulation of that type of wave. For each type of wave, use the number scrollers to change the amplitude, wavelength and frequency of the wave. Press the grid icon to the right of these scrollers to hide/show the grid. Press any of the particles to highlight them.

Press reset to return to the start screen and choose another wave type.
